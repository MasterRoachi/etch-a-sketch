# Etch-A-Sketch

A browser-based Etch-A-Sketch project built for [The Odin Project Foundations Course](https://www.theodinproject.com/lessons/foundations-etch-a-sketch).

This project uses JavaScript to generate a dynamic sketch grid. Users can draw by hovering over the grid squares, refresh the board, and choose a new grid size up to 100 squares per side.

## Live Preview

[View the live project](https://masterroachi.github.io/etch-a-sketch/)

## Features

* Default 16 × 16 drawing grid
* Dynamic grid generation with JavaScript
* Custom grid size prompt
* Maximum grid size of 100 × 100
* Hover-to-draw behavior
* Random RGB color effect
* Progressive darkening effect
* Responsive square sketch area
* Styled with a dark and gold Master Roachi theme

## Project Notes

The main goal of this project was to build a sketchpad-like browser toy without hardcoding the grid squares in HTML. The grid is created entirely through JavaScript, and each square responds to mouse hover events.

The project also includes the optional challenge features from The Odin Project: randomized colors and progressive darkening.
